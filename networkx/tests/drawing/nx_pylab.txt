nx_pylab
========

Drawing networkx graphs with matplotlib and graphviz.


>>> from networkx import *
>>> from networkx.drawing import *


>>> N=barbell_graph(5,10)

>>> hold(False)
>>> draw_spring(N)
>>> savefig("test.png")
>>> draw_random(N)
>>> savefig("test.ps")
>>> draw_circular(N)
>>> savefig("test.png")
>>> draw_spectral(N)
>>> savefig("test.png")

...    savefig("test.ps")
